# 🎓 UniAcademic - Sistema de Gestão Acadêmica

![UniAcademic Banner](https://img.shields.io/badge/UniAcademic-Premium%20Portal-F59E0B?style=for-the-badge)

O **UniAcademic** é uma plataforma moderna e intuitiva de gestão acadêmica, desenvolvida para proporcionar uma experiência fluida para alunos, professores e administradores. O sistema utiliza uma estética *glassmorphism* com modo escuro nativo e foco em alta performance.

## 🚀 Funcionalidades Principais

### Para Alunos 🧑‍🎓
*   **Dashboard Inteligente:** Resumo de CRA, disciplinas ativas e notificações recentes.
*   **Grade Semanal Interativa:** Visualização clara de horários com escala otimizada.
*   **Boletim Online:** Acesso rápido a notas e histórico de faltas.
*   **Integração com IA:** Suporte via Tutor de IA para auxílio nas disciplinas.

### Para Professores 👨‍🏫
*   **Gestão de Chamadas:** Sistema rápido e intuitivo para controle de presença.
*   **Lançamento de Notas:** Interface simplificada para gestão de avaliações.
*   **Plano de Aula Dinâmico:** Ferramentas para organizar e compartilhar materiais de aula.
*   **Cronograma Resumido:** Visão rápida das próximas aulas e status em tempo real.

### Administrativo ⚙️
*   **Analytics Institucional:** Gráficos de crescimento e métricas de matrículas.
*   **Gestão de Entidades:** Controle completo de alunos, professores, turmas e disciplinas.
*   **Logs de Auditoria:** Monitoramento de atividades em tempo real.

## 🛠️ Stack Tecnológica

*   **Frontend:** React.js com Tailwind CSS.
*   **Design:** CSS customizado com variáveis dinâmicas e animações suaves.
*   **Ícones:** Material Symbols Outlined.
*   **Gestão de Estado:** Context API da autenticação.
*   **Roteamento:** React Router v6 com separação modular por entidade.

## 📁 Estrutura do Projeto

*   `/src/pages/admin`: Telas de gestão do sistema.
*   `/src/pages/professor`: Ferramentas do docente.
*   `/src/pages/student`:Portal do aluno.
*   `/src/pages/auth`: Fluxos de login e registro.
*   `/src/routes`: Roteamento segregado para melhor escalabilidade.

## 💻 Como Rodar o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/UniAcademic/UniAcademic_Frontend.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
